

# LEGO RC Remote Car



This is the code for my RC car project! This project allows for ***ANYONE*** to controll an RC can from anywhere in the world while recieving  a live view from a camera!
It comes in 3 parts.
1. The hub: its what runs on the actual robot, communicating with my PC
2. The middleman (my PC): This one relays client controls.
3. The client: displays feed and also captures controlls
